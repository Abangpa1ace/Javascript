[requestAnimationFrame]

  - window.requestAnimationFrame(); : 애니메이션 최적화(부드럽게)
    * 이전엔 setInterval 사용 -> pixel 오류생김, 모바일 배터리 소모많음.

  - requestAnimationFrame(() => {
      console.log(0);
    })
    console.log(1);
      : 1 -> 0 순으로 출력(나중에 실행됨)

  - 기능: function render() 안에, requestAnimationFrame(render) 함수를 무수히 반복시킴. (60회/s 목표)
